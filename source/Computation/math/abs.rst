+++++++++++++
abs()
+++++++++++++

描述
=====
计算一个数的绝对值。

语法
=====
abs(x)

参数
====
x:接受到的数值

返回
====
返回x的绝对值

+--------+------+
|取值范围|返回值|
+========+======+
|x > 0   |x     |
+--------+------+
|x = 0   | 0    |
+--------+------+
|x < 0   | -x   |
+--------+------+

例程
=====


.. code:: c

	val1 = abs(val1);//求val1的绝对值，从而保证val1的值永远不为负
