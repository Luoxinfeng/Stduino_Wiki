min()
+++++++++++++

描述
=====
比较两个数字中的最小值

语法
=====
min(x,y)

参数
====
x:第一个数字（任何数据类型）

y:第二个数字（任何数据类型）










返回
====
两个数字中的较小者

例程
=====


.. code:: c

	int val1 =40;
	val1 = min(val1, 30);//将30与val1中最小的值赋给val1
