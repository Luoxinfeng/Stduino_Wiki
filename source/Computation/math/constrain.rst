+++++++++++++
constrain()
+++++++++++++

描述
=====
将一个数约束在一个固定范围内

语法
=====
constrain(x,a,b)

参数
====
x：要被约束的数字。

a：该范围的最小值。

b：该范围的最大值。



返回
====

+---------+------+
|取值范围 |返回值|
+=========+======+
|a < x < b|x     |
+---------+------+
|x <= a   | a    |
+---------+------+
|x >= b   | b    |
+---------+------+

例程
=====


.. code:: c

	val1 = constrain(10, val1, 30);//将val1的值限定在10到30之间。





注意
====
