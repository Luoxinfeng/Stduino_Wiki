+++++++++++++
sq()
+++++++++++++

描述
=====
求算术平方根

语法
=====
sq(num)

参数
====
num:整型或浮点类型数值（int/float）








返回
====
返回float类型

例程
=====


.. code:: c

	int num0 = 16;
	int num1 = 1;
	num1 = sq(num0);//将num开平方，并将算术平方根的值赋给num1
